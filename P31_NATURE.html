NATURE

The following paragraphs describe nature as being the set which contains all phenomena, all noumena, and all of pure nothingness.

A phenomenon is some finite region of space which is distinguishable from pure nothingness.

Phenomena is a set containing multiple elements such that each one of those elements is a exactly one distinct phenomenon.

An example of a phenomenon is the observation of the color red by human brains which are processing input signals from optic nerves detecting light whose wavelengths are approximately 700 nanometers. (Where the color red is being observed by some partial frame of reference is where other colors such as blue and green are not simultaneously being observed by that same partial frame of reference).

A noumenon is a specific phenomenon which is not being observed by some partial frame of reference (according to that particular frame of reference).

Noumena is a set containing multiple elements such that each one of those elements is a exactly one distinct noumenon.

If the many worlds hypothesis (i.e. the hypothesis which posits that one universe splits into multiple parallel universes when some decision is made in the parent universe) is true, then multiple partial frames of reference exist simultaneously (which means that one partial frame of reference may be rendering the color red as a phenomenon instead of rendering the color blue while some other partial frame of reference is simultaneously rendering the color blue as a phenomenon instead of rendering the color red (which means that, what one of those two partial frames of reference observes is a noumenon according to the other partial frame of reference)).

Pure nothingness is the ubiquitous and featureless substrate into which all phenomena are projected.

Randomness is the feature of a partial frame of reference not knowing what the outcome to a particular decision-making process (which that partial frame of reference observes) will be before that partial frame of reference observes the outcome to that decision-making process. An omniscient frame of reference does not experience randomness while that omniscient frame of reference is in its most macroscopic vantage.

A frame of reference is an allocation of pure nothingness into which zero phenomena, a natural number of phenomena, or infinitely many phenomena are projected.

A partial frame of reference does not render each phenomenon simultaneously.

An omniscient frame of reference simultaneously and immutably renders each phenomenon which each partial frame of reference is observing.
