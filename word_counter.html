WORD_COUNTER

1. Enter up to 3000 characters of text into the text field below.

2. Click the PROCESS button to get the total number of characters entered, the total number of words entered, and the number of times each unique word occurs inside the input string.

Note that a word, within the context of this application, is a string consisting solely of letters of the English alphabet, decimal digits, and the underscore character (_). Characters which are neither letters of the English alphabet nor decimal digits nor the underscore character are treated by this application as delimiter characters.
