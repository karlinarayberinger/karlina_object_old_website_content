KARBYTES_ETHICS

The following terms and their respective definitions describe the priorities of the author of this website.

This web page is an elaboration of concepts which are discussed in AGENCY and in KARBYTES_EXPERIENCE.

HEDONISM: goal-oriented behavior which an information processing agent implements in order for that information processing agent to generate as much satisfaction from its experiences as possible (and satisfaction is an information processing agent’s assessment that what it is presently experiencing is sufficiently desired by that information processing agent and not a waste of its cognitive resources).

Note that cognitive resources are the energy, time, and space which an information processing agent consumes in order to complete some goal-oriented task.

KNOWLEDGE: goal-oriented behavior which an information processing agent implements in order for that information processing agent to generate as extensive and detailed of a software map of reality as possible.

(The software map of reality which the author of this websites is experiencing is referred to by the term KARBYTES_EXPERIENCE within the context of this this website (and within the context of the object which is referred to by the term KARBYTES_EXPERIENCE)).

ART: the process of an information processing agent assimilating knowledge as a function of empirical data, deriving a sense of inspiration from that knowledge, and using that information processing agent’s output devices to manipulate objects in its environment in a manner which is maximally satisfying to that information processing agent.

Within the context of this web page, the term inspiration refers to the process of an information processing agent imagining forms which are satisfying to the information processing agent to instantiate as its own “proprietary” imaginary objects.

(A proprietor of a piece of information is assumed to be the first person to access that particular piece of information).
